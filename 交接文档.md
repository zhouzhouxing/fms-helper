# FMS Helper V2 — 功能交接文档

> 最后更新：2026-06-09

---

## 一、软件概述

FMS Helper 是一个 Electron 桌面应用，用于生成车载充电机（OBC/DCDC/逆变）的自动化测试脚本（CFG666格式）。

- **技术栈**：Electron + 原生 HTML/CSS/JS（零依赖、零构建工具）
- **项目路径**：`D:\脚本合集\Open code\fms-helper\`
- **打包产物**：`dist\FMS-Helper-1.0.0-Portable.exe`

---

## 二、支持的 8 种测试模式

| 芯片名称 | 内部值 | 包含的模块 |
|----------|--------|-----------|
| OBC 单相 | `obc-single` | OBC单相 |
| OBC 三相 | `obc-three` | OBC三相 |
| OBC 混合 | `obc-mixed` | OBC单相+三相（点位勾选区分） |
| DCDC | `dcdc-only` | DCDC单独 |
| 逆变 | `invert` | 逆变单独 |
| 单相+DCDC | `obc-single-dcdc` | OBC单相 + DCDC |
| 三相+DCDC | `obc-three-dcdc` | OBC三相 + DCDC |

## 八、核心计算规则

### 8.1 误差互斥逻辑（Offset vs Accuracy）

每个参数有2个输入：绝对误差(Error) + 精度%(Accuracy)
- 只填Error → 用绝对误差，输出 `OffsetMin=-Error`, `OffsetMax=Error`
- 只填Accuracy → 用精度，输出 `AccuracyMin=-Accuracy%`, `AccuracyMax=Accuracy%`
- **两个都填** → 计算实际范围：`Error范围=Error值`，`精度范围=基准值×Accuracy%`，取**范围更大的**
- 都不填 → 用默认绝对误差（电压5，电流0.5）

### 8.2 功率计算

- **OBC输入功率**：`MachOBCInPowerMin/Max = 输入电压Min/Max × 输入电流Min/Max`（电流<0.1按0.1算）
- **OBC输出功率**：`MachOBCOutPowerMin/Max = 输出电压Min/Max × 输出电流Min/Max`（同上）
- **DCDC**：字段名换成DCDC，逻辑相同
- **逆变输入功率**：用OBC字段名，`MachOBCInPower = 输入电压(DC) × 输入电流(DC推算)`
- **逆变输出功率**：`MachOBCOutPower = 输出电压(AC) × 输出电流(AC)`

### 8.3 电流推算

- **OBC输入电流**：`Iin = Pout / (效率/100) / Vin`（用户不填时自动推算）
- **OBC输出电流**：`Iout = Pout / Vout`（用户不填时自动推算）
- **三相相电流**：`Iphase = Pout / (效率/100) / 相电压 / 3`
- **DCDC**：同OBC逻辑
- **逆变输入电流(DC)**：`Iin = Pout / (效率/100) / Vin(DC)`

### 8.4 量程自动推算

从档位列表 `[15,30,60,100,150,300,600,1000]V` 和 `[0.5,1,2,5,10,20,40,50]A` 中，取**大于等于（实际值+裕量）** 的最小档位：
- 电压裕量：+20%（小电压精度高，大电压包得住波动）
- 电流裕量：+50%（小电流精度高，大电流包得住波动）
- 量程值格式：`300V`、`20A`、`500mA`（0.5A）
- 取所有点位中的最大值，全局只生成1个量程步骤

### 8.5 逆变负载值计算

| 项目 | CR模式 | CC模式 |
|------|--------|--------|
| 预拉载LoadValue | 输出电压÷5（截断1位小数） | 5 |
| 拉载LoadEndValue | 输出电压÷输出电流（截断1位小数） | 输出电流 |
| 降载LoadEndValue | 输出电压÷1（截断1位小数） | 1 |

## 九、联动规则

| 触发 | 联动结果 |
|------|---------|
| DC输出端="连接" | 上辅电写 `DCOutPoint=连接`，关辅电写 `DCOutPoint=断开`。不选则都不写 |
| KL15="打开" | 上辅电写 `EnableKL15=打开`，关辅电写 `EnableKL15=关闭`。不选则都不写 |
| 48V继电器="开" | 预拉载写 `DCOutSwitchRelay=开` + `DCOutRelay=关`，关载写 `DCOutSwitchRelay=关`。不选则都不写 |
| 逆变DC输出端="连接" | 辅电写 `DCOutPoint=连接`，关辅电写 `DCOutPoint=断开`。不选则都不写 |
| 三相模式下输入电压类型="线电压" | HVAC步骤AcVolt = Vin ÷ √3（取整） |
| 三相模式下输入电压类型="相电压" | HVAC步骤AcVolt = Vin（不转换） |

## 十、用户填写规则

| 规则 | 说明 |
|------|------|
| 不填就不写 | 用户不填的字段，输出CFG666时**不包含该字段**，项目工具会用默认值 |
| 必填 vs 可选 | 必填字段为空时生成会报错提示 |
| 序号递增 | 只有带FuncDesc的步骤才消耗序号，纯操作步骤（延时、关载等）不占序号 |
| 多点位量程 | 多个点位只生成1个量程步骤，取所有点位的最大值 |
| 逆变只写单相 | 目前逆变不支持三相 |
